# Rd Veiculos API

Reposítório contendo uma api desenvolvida com .Net Core juntamente com o projeto de testes unitários e teste de integração, que está dentro da pasc Testes/Rd.Veiculos.Tests.Integration.

## Pré-requisitos
- .NET Core 6
- Sql Server

## Instalação
1. Clone este repositório.
2. Execute `dotnet restore` para restaurar as dependências.

## Banco de Dados
- Execute o script SQL contido na pasta "sql" para criar o banco de dados.

## Documentação C4
- A documentação C4 está disponível na pasta "arquitetura".

## Como Executar
1. Navegue até a pasta "src".
2. Execute `dotnet build` para compilar o projeto.
3. Execute `dotnet run` para iniciar a aplicação.
